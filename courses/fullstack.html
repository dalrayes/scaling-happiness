Caravan Full Stack Immersive

Build the skills to become a software developer

Caravan's Immersive course is a 13 week program designed to prepare you as a junior developer. The program will cover techniques including programmatic thinking, full stack javascript development and key soft skills. We use the following languages and frameworks: Javascript, HTML/CSS, jQuery, AJAX,SQL & no SQL, nodeJS, expressJS, mongoDB, MEAN React JS and more.

Fundamentals

  • Design
  • Programmatic Thinking
  • Javascript fundamentals

Immersive

  • Javascript deep dive
  • Databases & Deployment
  • Client & Server-Side
  • Build an app with teams

  • Personal branding & online presence
  • Mock interviews
